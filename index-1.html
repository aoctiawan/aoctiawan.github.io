Aplikasi ini menggunakan framework Codeigniter versi 2 dan di upgrade ke versi 3, dengan MySQL sebagai Database.

JPT Pratama (Jabatan Pimpinan Tinggi Pratama) adalah aplikasi lelang jabatan yang digunakan oleh pegawai yang berminat untuk menduduki Jabatan Pimpinan Tinggi yang berada di suatu instansi / satuan kerja yang ada di lingkungan Kementerian Sosial dengan mengajukan dirinya untuk kemudian dibandingkan dengan kualitas calon pimpinan tinggi yang lain.

Aplikasi ini menggunakan framework Laravel versi 7 (yang kemudian di upgrade ke versi 9), dengan MySQL sebagai Database.

JDIH (Jaringan Dokumen dan Informasi Hukum)
adalah wadah pendayagunaan bersama atas dokumen hukum secara tertib, terpadu, dan berkesinambungan, serta merupakan sarana pemberian pelayanan informasi hukum secara lengkap, akurat, mudah, dan tepat (PM Kominfo No. 20 Tahun 2013 tentang Jaringan Dokumentasi dan Informasi Hukum Kementerian Kominfo).

Aplikasi ini menggunakan framework Yii, dengan MySQL sebagai Database.

Stunting adalah sistem yang digunakan sebagai dashboard untuk peta penyebaran stunting dan informasi-informasi mengenai stunting.

Aplikasi ini menggunakan framework Laravel versi 7 (yang kemudian di upgrade ke versi 9), dengan SQL Server sebagai Database, dan menggunakan Leaflet untuk library GIS.

SIKD-API adalah Restful API yang digunakan sebagai backend dari aplikasi mobile dari SIKD (Sistem Informasi Kearsipan Dinamis), SIKD sendiri adalah sistem persuratan yang digunakan oleh Kementerian Sosial.

Aplikasi ini menggunakan framework Laravel, dengan My SQL sebagai Database.

Anggaran adalah aplikasi yang digunakan oleh Pusat Data dan Informasi Kesejahteraan Sosial (Pusdatin Kesos) Kementerian Sosial untuk memasukkan data seperti Perjadin, Fullboard, dan Fullday.
